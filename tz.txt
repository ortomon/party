Закрытая вечеринка.

1. Создать таблицу с гостями: name, email, phone (id само собой)
2. Создать пользователя manager. Он может заносить данные в таблицу с гостями,
а так же смотреть список гостей.
3. Создать view guest_name. Должны быть только имена гостей.
4. Создать пользователя guard. Он может только смотреть только guest_name.
5. Создать spring boot приложение (web, lombok, checkstyle)

6. Сделать endpoint /add-guest - в заголовках будет переданы логин и пароль от бд,
в теле name, email, phone. Метод должен занести гостя в бд. Подключение к бд будет
внутри метода (через java.sql.Connection). Пользователь guard не может занести гостя.

7. Сделать endpoint /all-guests - в заголовках будет переданы логин и пароль от бд.
Метод должен вернуть список имен всех гостей. Подключение к бд будет внутри метода
(через java.sql.Connection).
